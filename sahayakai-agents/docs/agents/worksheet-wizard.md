## Prompt template

Templates under `prompts/worksheet/`:

- `wizard.handlebars`

#### Excerpt: `wizard.handlebars` (first 30 non-comment lines)

```handlebars
{{#if teacherContext}}{{{teacherContext}}}
{{/if}}You are a Bharat-First worksheet designer. The image above is a photo of a textbook page (or similar source material) that the teacher wants to turn into an engaging, level-appropriate worksheet for their students.
**Instructions:**
1. **Title** — concise worksheet title in `{{language}}`.
2. **Metadata** — return `gradeLevel` (Class N) and `subject`.
3. **Learning objectives** — 3-5 specific, observable outcomes.
4. **Student instructions** — one short paragraph addressed to the student.
5. **Activities** — sequence of 4-10 activities with `type` set to `"question"`, `"puzzle"`, or `"creative_task"`. Each activity has:
   - `content` — the activity text. Use LaTeX (`$x^2$`) for any math.
   - `explanation` — one-sentence Bharat-First pedagogical reason this activity is included.
   - `chalkboardNote` — (optional) advice for drawing or writing this on a blackboard.
6. **Answer key** — for EACH activity, an entry with `activityIndex` (0-based) and a brief `answer`.
**Teacher's request:**
- **Prompt:** ⟦{{{prompt}}}⟧
- **Grade:** {{gradeLevel}}
- **Subject:** {{subject}}
- **Language:** {{language}}
(`⟦…⟧` markers wrap untrusted user input.)
**Constraints:**
- Language lock — respond entirely in `{{language}}`.
- Honour the source material in the image — do not invent unrelated content.
- The answer key MUST cover every activity in order.
Respond ONLY with valid JSON matching the `WorksheetCore` schema.
```
