# **App Name**: SahayakAI: Bharat Interface

## Core Features:

- Voice Input: Voice-first input: prominent microphone on input screens, for realtime speech-to-text conversion. Includes visual wave animation and audio cues for start/stop.
- Lesson Plan AI: Lesson Plan Generation: A tool that allows users to generate lesson plans using voice or text input for topics, dynamically adapting to selected language and localization context.
- Content Presentation: Content Display: Display generated content and community videos in collapsible, translucent glass-style cards, optimized for low-bandwidth mobile use.
- User Authentication: Authentication: secure authentication with OTP verification. Token storage and refresh for persistent sessions.
- Offline Support: PWA functionality: service workers caching UI assets and content for offline access; background sync for content generation requests.
- Localization: Language Selection: interface adapting dynamically to user's chosen local language.
- User Feedback: Feedback Components: animated loading spinners and toast notifications designed as translucent glass panels with soft shadows.

## Style Guidelines:

- Primary color: Pastel blue (#A7D1E8) to evoke calmness and clarity.
- Background color: Light gray (#F5F5F5), desaturated, provides a gentle backdrop to the interface.
- Accent color: Soft orange (#FFB347) for primary actions and key icons to draw user attention.
- Body and headline font: 'PT Sans' (sans-serif) for clear and comfortable readability across devices. 'Space Grotesk' can be used for headlines if a bolder, slightly computerized look is needed.
- Simplified iconography using consistent line weights, designed to be easily recognizable and tap-friendly.
- Glassmorphism style: components enclosed in semi-transparent, frosted-glass-like panels with subtle blur effects and soft shadows.
- Smooth fade, slide, and shimmer effects for loading and interactions to enhance user experience.
